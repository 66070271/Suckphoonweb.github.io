About the Project

ข้อมูลเกี่ยวกับโครงการตรวจสอบคุณภาพอากาศ...

Project ของเราได้มีการทดลองการใช้งานจริงโดยใช้เขม่าไฟจากธูปเพื่อให้เห็นว่าการวัดค่าฝุ่นนั้นเป็นตามความจริงและเกิดErrorไม่มากนัก

เราได้มีการนำ LCD มาแสดงผลค่าฝุ่นออกมาแบบ Realtimeโดยรับค่ามาจาก Sensor โดยตรง

Project ของเราจะมีการส่งค่าผ่านตัว Sever MQTT

จากนั้นเราจะนำข้อมูลที่เราดึงจาก Topic ใน Sever ที่เราได้รับค่านั้นมาประมวลผลและแสดงผล
